# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-08

### Added
- Initial release of Fabric Spark Application
- Scala implementation with Maven build support
- Python (PySpark) implementation
- DataProcessingApp main application class
- DataTransformer utility for data transformations
- DataValidator utility for data validation
- Sample data and configuration files
- Build scripts for automated building
- Run scripts for both Scala and Python versions
- Comprehensive documentation (README, DEPLOYMENT, DEVELOPMENT)
- Support for CSV and Parquet formats
- Logging configuration with log4j
- YAML-based application configuration
- .gitignore for Spark/Scala/Python projects
- MIT License
- Requirements file for Python dependencies

### Features
- Read data from CSV files
- Data cleansing and transformation
- Null value handling
- Metadata column addition
- Analytics and aggregation
- Write processed data to Parquet format
- Sample data generation for testing
- Date-based filtering
- Deduplication capabilities
- Data quality validation
- Column name standardization

### Documentation
- Complete README with usage instructions
- Deployment guide for multiple platforms
- Development guide for contributors
- Code examples and best practices
- Configuration documentation
